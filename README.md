### Installation:

**1** - Clone the repository
```bash
git clone https://github.com/Cequallium/Canvas.git && cd Canvas
```
**2** - Then install the dependencies
```bash
npm i
```
**3** - Run dev server 
```bash
npx vue-cli-service serve
```
